Troubleshooting Guide

Country/Onion.db Error

If you see errors related to countries or onion database:

  1. Download the onion.db file: onion.db
  2. Place it in: C:\Users\[YourUsername]\AppData\Local\Cawokvpthn\
  3. Restart Cawokvpthn VPN

Slow Internet Connection

If Tor connection is slow:

  1. Go to SETTINGS → BRIDGES
  2. In Bridge type select: snowflake
  3. Check the box as shown in the guide
  4. Reconnect to Tor
  5. If still slow, try obfs4:
    • Get bridges from @GetBridgesBot or Tor website
    • Paste bridges below and click Add
    • Reconnect
  6. If still slow, try webtunnel:
    • Get webtunnel bridges from @GetBridgesBot
    • Paste and click Add
    • Reconnect

Browser Landing Page Issue

If browser opens with stub page after connection:

  1. Go to Settings → Landing Pages
  2. In Connected page enter: https://framklenholknovitch.github.io/CawokvpthnRls/
  3. Or disable analytics by turning off the toggle
  4. Close settings and reconnect to Tor

DNS & Connection Issues

If Tor won't connect or connection is unstable:

  1. Enable DNS in settings - Tor often requires DNS to work
  2. Recommended DNS settings:
    • Google DNS: 8.8.8.8 and 8.8.4.4
    • Cloudflare DNS: 1.1.1.1 and 1.0.0.1
  3. Other DNS providers may be unstable
  4. This also improves connection speed

General Tips

  • Always restart the application after changing settings
  • Check your firewall/antivirus settings
  • Try disconnecting and reconnecting
  • Clear Tor cache if problems persist
  • Ensure you have the latest version
